# assembly-basics

Ensino desde o básico ao avançado em **ASSEMBLY** (x86_64, intel)

Únicos requisitos:

1. Uma máquina x86_64 (padrão para pcs pessoais com intel/amd)
2. Nasm instalado:

```
# Debian, Ubuntu e derivados
sudo apt install nasm

# Fedora, RHEL, almalinux e derivados:
sudo dnf install nasm

# Arch linux e derivados
sudo pacman -S nasm
```

3. Muita paciência e masoquismo

Como assembly é mais complexo, você geralmente irá precisar consultar, aconselho fortemente que faça anotações em um caderno ou no app Obsidian/Notion
